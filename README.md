# Dishes Form

## Initial setup

### Necessary components & software: 
  - npm ( used `7.19.1` )
  - NodeJS ( used `v14.18.0` )

### How to run project

- how to run the first time:
  - `npm install`
  - `npm start`
- how to run it on subsequent runs
  - `npm start`

### How to use the project

- what URL to open: 
  - `http://localhost:3000`
